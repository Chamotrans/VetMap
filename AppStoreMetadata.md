## Description (zh-Hant-HK)

VetMap 係一個專為台灣同香港毛孩家長設計嘅獸醫診所地圖 App。

**搵診所**
- 地圖模式瀏覽附近獸醫診所
- 關鍵字搜尋 + 地區/價格/驗證篩選
- 診所詳情：電話、網站、導航、營業時間

**睇評價**
- 真實用戶評價同評分
- 上傳診所相片
- 「有用」投票系統

**費用透明**
- 治療報價分享系統
- 預估費用 vs 實際費用
- TWD / HKD 雙幣支援

**社群貢獻**
- 新增診所：地址地理編碼
- 新增評價：星星評分 + 相片上傳
- 報價分享：13 種治療類型

**Premium 會員**
- 完整診所資料庫
- 無廣告體驗
- 優先客戶支援

**寵物好物**
- 食品、玩具、保健、藥品瀏覽
- 香港台灣雙市場產品覆蓋

**寵物保險**
- 台灣 3 間 + 香港 3 間保險比較
- 保費、保障範圍、排除條款一覽

VetMap — 願世間再無誤診毛孩。
